## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Data Management

All tools and stacks are stored in local JSON files:
- `src/data/tools.json` - Tool directory
- `src/data/stacks.json` - Tech stack collections

To add or modify tools, simply edit these JSON files and rebuild. No database or external API required!

## Tech Stack

- React 19 + TypeScript
- Vite for fast builds
- Chakra UI for components
- React Router for navigation
- Deployed on Netlify
